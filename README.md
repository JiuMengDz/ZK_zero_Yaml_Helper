# **ZK-zero Yaml Helper YAML导表配置文件辅助**
`[公司内部使用]` 导表配置文件.yaml的可视化编辑辅助工具

## 使用教程
1. 打开`.yaml`后缀名的文件
2. `Ctrl + Shift + P`
3. 选择 `View: Reopen With...`
4. 选择 `YAML HELPER`
